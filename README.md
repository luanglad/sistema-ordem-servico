# Sistema de Ordem de Serviço

Sistema web para gerenciamento de ordens de serviço de uma assistência técnica especializada em eletrônicos (videogames).

## Tecnologias
- Python 3.11
- Django
- SQLite

## Objetivo
Projeto desenvolvido para uso real em ambiente local, com foco em organização, segurança e controle completo do ciclo de ordens de serviço.
